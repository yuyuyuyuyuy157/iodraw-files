```mermaid
graph TD
    Start([开始]) --> Init[初始化图形界面，初始化多项式A和B]
    Init --> Menu{主菜单选择}
    
    Menu -->|1: 创建/重新输入多项式| Create[输入多项式A和B]
    Create --> Menu
    
    Menu -->|2: 多项式相加| Add[计算 A + B]
    Add --> OutputAdd[显示结果]
    OutputAdd --> Menu

    Menu -->|3: 多项式相减| Sub[计算 A - B]
    Sub --> OutputSub[显示结果]
    OutputSub --> Menu

    Menu -->|4: 多项式相乘| Mul[计算 A * B]
    Mul --> OutputMul[显示结果]
    OutputMul --> Menu

    Menu -->|5: 多项式求值| EvalMenu{求值菜单选择}
    
    EvalMenu -->|1: A(x)| EvalA[输入x值，计算A(x)]
    EvalA --> OutputEvalA[显示结果]
    OutputEvalA --> EvalMenu

    EvalMenu -->|2: B(x)| EvalB[输入x值，计算B(x)]
    EvalB --> OutputEvalB[显示结果]
    OutputEvalB --> EvalMenu

    EvalMenu -->|3: (A+B)(x)| EvalAdd[输入x值，计算(A+B)(x)]
    EvalAdd --> OutputEvalAdd[显示结果]
    OutputEvalAdd --> EvalMenu

    EvalMenu -->|4: (A-B)(x)| EvalSubEval[输入x值，计算(A-B)(x)]
    EvalSubEval --> OutputEvalSub[显示结果]
    OutputEvalSub --> EvalMenu

    EvalMenu -->|5: (A*B)(x)| EvalMul[输入x值，计算(A*B)(x)]
    EvalMul --> OutputEvalMul[显示结果]
    OutputEvalMul --> EvalMenu

    EvalMenu -->|6: 返回| Menu

    Menu -->|6: 退出程序| End([结束])
    Menu -->|其他输入| Error[无效输入提示]
    Error --> Menu

```
